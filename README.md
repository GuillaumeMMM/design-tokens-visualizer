# Design tokens visualizer

Visualization tool for design tokens based on (part of) the [DTCG](https://www.designtokens.org/tr/2025.10/format/) (Design Tokens Community Group) specifications. Accessible on [tokens-visualizer.guillaumemeigniez.me](https://tokens-visualizer.guillaumemeigniez.me).
